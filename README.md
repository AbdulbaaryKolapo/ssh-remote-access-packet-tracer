<h1>ssh-remote-access-packet-tracer</h1>


<h2>Description</h2>
This project demonstrates how to configure and test SSH for secure remote access in a Cisco Packet Tracer network simulation.

<br />


<h2>Languages and Utilities Used</h2>

- <b>Cisco Packet Tracer CL</b> 
  

<h2>Environments Used </h2>

- <b>Windows 11</b> (21H2)

<h2>Steps to do to achieve this:</h2>

- <b>Set up a basic network topology (router, switch, and one or more PCs).</b>
- <b>Configured IP addressing.</b>

- <b>Enabled SSH on the router or switch:</b>

  - <b>Set hostname and domain name.</b>

  - <b>Created a local user with a password.</b>

  - <b>Generated RSA keys.</b>

  - <b>Enabled the VTY lines for SSH access.</b>

- <b>Used a PC in the simulation to initiate an SSH session to the router using the terminal</b>
